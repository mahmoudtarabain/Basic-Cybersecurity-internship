# Basic-Cybersecurity-internship
Cybersecurity internship tasks and demos.

# Description
This repository holds all the tasks and demos from my Basic Cybersecurity Internship.
It’s designed to help you understand core cybersecurity concepts like the CIA Triad, encryption methods, digital signatures, and basic network security.

## Documentation (PDFs)
01-CIA_Triad.pdf – Simple explanation of the CIA Triad
02-Task2_Symmetric_Encryption.pdf – Overview of symmetric encryption
03-Task3_Asymmetric_Encryption.pdf – Overview of asymmetric encryption
04-Task4_Digital_Signatures.pdf – Introduction to digital signatures
05-Task5_Network_Security.pdf – Basics of network security

## Python Scripts (Demos)
06-symmetric_encryption_demo.py – Hands-on demo of symmetric encryption
07-asymmetric_encryption_demo.py – Hands-on demo of asymmetric encryption
08-digital_signatures_demo.py – Hands-on demo of digital signatures

## How to Run the Scripts
 1. Install Python (version 3.8 or higher recommended)
 2. Open terminal or command prompt
 3. Navigate to the project folder
 4. Run any script using: bash
python filename.py
